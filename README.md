# Job Hunt

Completed and deployed **Job Hunt**, a full-stack MERN application for job discovery and application management.

## 🚀 Key Features

*   **Frontend**: Developed a highly responsive and modern UI using React (Vite)
*   **Backend built on RESTful APIs**: Powered by Node.js and Express
*   **Database**: Integrated MongoDB Atlas for secure, cloud-based data storage
*   **Secure Authentication**: Implemented user authentication using JWT alongside protected routes
*   **Media Handling**: Utilized Cloudinary for handling media and file uploads seamlessly
*   **Cloud Deployment**: Deployed the frontend on Vercel and the backend on Render
